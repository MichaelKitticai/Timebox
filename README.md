# TimeBox

Are you always running late to meetings or events and wondering how to squeeze a few extra minutes out of your day? Worry no more! TimeBox is here to help you stay on track and arrive on time. This simple web app calculates your schedule based on your event time, prep time, and commute so you can plan your day with ease.

## Features
- Input your event details and get a clear, step-by-step timeline.
- Choose from quick, normal, or relaxed presets.
- Save your favorite settings for future use.
- Installable as a PWA to use on your phone.

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the development server with `npm run dev`.

Enjoy staying on time with TimeBox!
